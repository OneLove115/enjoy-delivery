# Architecture

Overview
Geolocation-first food delivery platform with auto currency and real-time tracking.

Frontend
- React Native
- Next.js

Backend
- Node.js + Express or FastAPI
- REST + WebSockets

Core Services
- PostgreSQL
- Redis

Integrations
- Google Maps or Mapbox
- Stripe and iDEAL
- Currency API

Infrastructure
- Vercel
- AWS or Railway

Flow
1. Detect location
2. Fetch restaurants
3. Place order
4. Assign driver
5. Track delivery
